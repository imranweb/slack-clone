# Simple Chat Application

## Build applicaiton
yarn build 

## Test applicaiton
yarn test 


## Deployment
CircleCI and Heroku is being used as CI/CD

## Tech Stack
* NodeJS
* Webpack
* Express
* ReactJS
* Redux
* Redux Saga
* Material UI
* Pusher Chatkit
* Circle CI with Heroku
* Chai, Mocha, Enzyme
